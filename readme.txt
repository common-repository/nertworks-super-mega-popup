== Description ==

Super Mega Popup makes it easy to place popups on a particular page, post or site wide.  Offers different options for user to select to customize their popup.  Newly added.  Popups can be edited just like a Wordpress Post.  

== Installation ==
1. Upload `nertworks-super-mega-popup/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
